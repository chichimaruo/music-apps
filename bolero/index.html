「楽器リレー」＋「クレッシェンド」をONにすると、ppp からだんだん大きくなる原曲の体験ができます。図のたての位置は音の高さです。

原曲ではどの楽器が演奏している？（登場順）

旋律A：①フルート → ②クラリネット → ⑤オーボエ・ダモーレ → ⑥フルート＋弱音器トランペット → ⑨ホルン＋チェレスタ → ⑩木管の四重奏 → ⑬⑭ヴァイオリン＋木管 → ⑰総奏
旋律B：③ファゴット → ④小クラリネット → ⑦テナーサックス → ⑧ソプラノサックス → ⑪トロンボーン → ⑫高音の木管 → ⑮⑯金管も加わる → ⑱全楽器＋転調！
